# FITTING 拖拽组件页面设计

FITTING既是一个学习VUE的教程，有是一个可以通过拖拽组件的可视化设计框架，它将可以自行设计你的网站的功能，甚至你可以完全不会开发，而要展示你的网页只需要拖拖拖。
